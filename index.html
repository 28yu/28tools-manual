28 Tools

Revit 2021 アドイン操作マニュアル

🔘 符号 ON/OFF

通り芯・レベルの符号表示をワンクリックで切り替える機能です。選択している通り芯がある場合は選択要素のみ、未選択時は現在のビューの全ての通り芯・レベルに影響します。

⚏
両端 - 通り芯・レベルの両端に符号を表示/非表示
⚏
左端 - 通り芯・レベルの左端（下端）のみに符号を表示/非表示
⚏
右端 - 通り芯・レベルの右端（上端）のみに符号を表示/非表示

使用方法

  1. 符号を変更したい通り芯・レベルを選択（任意）
  2. 28 Toolsタブの「符号 ON/OFF」パネルから目的のボタンをクリック
  3. 即座に符号表示が切り替わります
💡 ポイント: 選択なしの場合は現在のビューの全要素、選択ありの場合は選択要素のみに適用されます。

📋 シート一括作成

指定した図枠を使用して複数のシートを一括作成する機能です。最近使用した図枠が優先表示され、作成数やプレフィックスも選択できます。

📋
一括作成 - 図枠・作成数・プレフィックスを選択してシートを一括作成

使用方法

  1. 「一括作成」ボタンをクリック
  2. 図枠ファミリを選択（最近使用した図枠が上位表示）
  3. 図枠タイプを選択
  4. 作成数を選択（1枚、5枚、10枚、20枚）
  5. プレフィックスを選択（なし、平面図、詳細図、断面図）
  6. 「作成実行」で一括作成開始
💡 ポイント: シート番号は既存シートの次の番号から自動採番されます。設定は記憶され、次回使用時に引き継がれます。

👁 3D視点コピペ

3Dビューの視点（カメラ位置・向き・ズーム）を他の3Dビューに一括コピーする機能です。プロジェクトブラウザでの複数選択にも対応しています。

👁
視点コピー - 現在アクティブな3Dビューの視点情報をメモリに保存
👁
視点ペースト - 保存した視点情報を選択した3Dビューまたはアクティブビューに適用

使用方法

  1. コピー元の3Dビューをアクティブにして「視点コピー」をクリック
  2. プロジェクトブラウザでコピー先の3Dビューを選択（Ctrl+クリックで複数選択可能）
  3. 「視点ペースト」をクリックして一括適用
💡 ポイント: ビュー未選択時はアクティブな3Dビューにペーストされます。画面更新が自動で行われ、即座に結果を確認できます。

📦 切断ボックスコピペ

3Dビューの切断ボックス設定（範囲・ON/OFF状態）を他の3Dビューに一括コピーする機能です。複数のビューで同じ切断範囲を統一できます。

📦
切断ボックスコピー - 現在アクティブな3Dビューの切断ボックス設定をメモリに保存
📦
切断ボックスペースト - 保存した切断ボックス設定を選択した3Dビューまたはアクティブビューに適用

使用方法

  1. コピー元の3Dビューをアクティブにして「切断ボックスコピー」をクリック
  2. プロジェクトブラウザでコピー先の3Dビューを選択（複数選択可能）
  3. 「切断ボックスペースト」をクリックして一括適用
💡 ポイント: 切断ボックスのON/OFF状態も含めて完全にコピーされます。画面更新が自動で行われ、即座に結果を確認できます。

🖼 ビューポート位置コピペ

シート上のビューポート位置を他のシートに一括コピーする機能です。自動マッチング機能により、同名ビューや類似ビューを自動で検出してコピーします。

🖼
ビューポート位置コピー - 選択したビューポートまたはシートの位置情報をメモリに保存
🖼
ビューポート位置ペースト - 保存した位置情報を選択したシートのビューポートに自動マッチングで適用

使用方法

  1. コピー元のビューポートを選択するか、シートを選択して「ビューポート位置コピー」をクリック
  2. プロジェクトブラウザでコピー先のシートを選択（複数選択可能）
  3. 「ビューポート位置ペースト」をクリックして自動マッチング実行
💡 ポイント: 完全一致・キーワードマッチング・階層マッチングの3段階で自動的にビューを照合します。複数シート選択時は自動処理されます。

✂ トリミング領域コピペ

ビューのトリミング領域設定を他のビューに一括コピーする機能です。平面図・断面図・立面図・詳細図・3Dビューなど幅広いビュータイプに対応しています。

✂
トリミング領域コピー - 現在アクティブなビューのトリミング領域設定をメモリに保存
✂
トリミング領域ペースト - 保存したトリミング領域設定を選択したビューまたはアクティブビューに適用

使用方法

  1. コピー元のビューをアクティブにして「トリミング領域コピー」をクリック
  2. プロジェクトブラウザでコピー先のビューを選択（複数選択可能）
  3. 「トリミング領域ペースト」をクリックして一括適用
💡 ポイント: トリミング領域のON/OFF状態も自動制御されます。コピー元がOFFの場合はペースト先もOFFに、ONの場合は自動的にONにして領域をコピーします。

動作環境

対応ソフトウェア Autodesk Revit 2021
対応OS Windows 10 / Windows 11
必要な権限 管理者権限（インストール時のみ）
ネットワーク インターネット接続（マニュアル閲覧時のみ）
ディスク容量 約1MB（軽量設計）
💡 注意事項:
  • Revit 2021専用です。他のバージョンでは動作しません。
  • アドインの動作にはRevitの有効なライセンスが必要です。
  • ネットワーク版Revitでも動作します。

インストール方法

🔧 自動インストール（推奨）

  1. 配布パッケージをダウンロードして展開
  2. 「自動インストール.bat」を右クリック
  3. 「管理者として実行」を選択
  4. Revitを再起動
  5. 「28 Tools」タブが表示されることを確認

📁 手動インストール

  1. 以下のフォルダを開く：
    C:\ProgramData\Autodesk\Revit\Addins\2021\
  2. 配布パッケージ内の以下のファイルをコピー：
    • Tools28.dll
    • Tools28.addin
  3. Revitを再起動

🔍 インストールできない場合

  • 「診断ツール.bat」を実行してエラー内容を確認
  • Revit 2021がインストールされているか確認
  • 管理者権限でコマンドプロンプトから実行
  • ウイルス対策ソフトがファイルをブロックしていないか確認

アンインストール方法

🗑️ 自動アンインストール（推奨）

  1. 配布パッケージ内の「アンインストール.bat」を右クリック
  2. 「管理者として実行」を選択
  3. 削除確認メッセージで「Y」を入力
  4. Revitを再起動
  5. 「28 Tools」タブが消えていることを確認

📁 手動アンインストール

  1. 以下のフォルダを開く：
    C:\ProgramData\Autodesk\Revit\Addins\2021\
  2. 以下のファイルを削除：
    • Tools28.dll
    • Tools28.addin
  3. Revitを再起動
💡 完全削除について:

アンインストール後、設定ファイルやログファイルは自動的に削除されます。手動で削除する必要はありません。
